# AI-Agents-using-Azure-AI-Foundry
Comprehensive project repository for mastering AI Agents with Azure AI Agent Service. Includes setup and demos for Azure AI Foundry, Assistants API, Action Tools (Function Calling, Code Interpreter), Knowledge Tools (Bing, File, Azure AI Search), and Python SDK implementations for real-world AI automation.

This project covers:

- **Azure AI Foundry**: Setup, architecture, hubs, projects, and playgrounds.
- **Assistants API**: Refresher on architecture, components, and function calling.
- **Azure AI Agent Service**: Agent creation, architecture, region support, and pricing.
- **Action Tools**:
  - Function Calling
  - Code Interpreter
- **Knowledge Tools**:
  - Bing Search
  - File Search
  - Azure AI Search
- **Azure SDK for Python**: Sample code for creating and managing AI agents.

## Key Features

- Step-by-step guidance for beginners and intermediate users.
- Hands-on demos with Python code.
- Real-world use cases for AI automation.
- Integration of no-code and Python solutions for Action & Knowledge tools.
